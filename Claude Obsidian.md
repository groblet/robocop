## 🎯 Bottom Line

Claude Obsidian is a custom skill for Claude Code that builds a persistent, compounding wiki inside Obsidian to give the AI long-term memory across sessions while minimising token costs.

## 📝 Key Takeaways

- 🧠 **Persistent Memory**: The skill creates a compounding digital second brain, ensuring Claude retains knowledge and grows smarter with each subsequent chat session.
- 📂 **Three-File Architecture**: The framework uses a `hot.md` file for immediate session context, an `index.md` file for a full map of pages, and individual wiki pages to store detailed data efficiently.
- 💰 **Token Efficiency**: By querying only the relevant index and files based on keyword specifics rather than the full history, it drastically reduces token consumption.
- 🛠️ **Simple Commands**: Users can use `/save` to store chat data and update the index, `/arch` to trigger deep automated research that populates the wiki, and `/canvas` to generate interactive visual boards.
- 📊 **Visual Analytics**: The canvas feature allows Claude to automatically position nodes, build flowcharts, generate graphs, and organise media like images, videos, and animations.
- 🚀 **Easy Installation**: The skill can be deployed quickly into a terminal using two installation commands from the official GitHub repository.
