**BLUF:** Google Android Halo is an upcoming status bar feature allowing background AI agents to update, question, and show results to users seamlessly.

📌 **CORE FUNCTION**
* 🧠 **Agent interface:** Acts as a dedicated space in the status bar for AI agents like Gemini to communicate.
* ⚙️ **Background tasks:** Enables agents to run multi-step tasks without interrupting the current user activity.
* 💬 **User interaction:** Surfaces follow-up questions, progress updates, and final results directly at the top of the screen.

🔒 **TECHNICAL ARCHITECTURE**
* 📦 **Containerised environment:** Utilises a virtual window system where the agent and the designated app are isolated.
* 🛡️ **Security restriction:** Prevents the AI agent from leaving the container or accessing unauthorised apps.
* 🔄 **System shift:** Marks Google's transition from a standard operating system to an action-driven intelligent system.
